# F1 - 几何图形生成器

一个基于文字命令的几何图形绘制工具，适合手机操作，无需鼠标交互。

## 功能特点

- 支持多种几何图形：矩形、直线、圆、点
- 支持相对位置计算：中点、分点（1/3、2/3等）
- 多种颜色选择：红、蓝、绿、黄、紫、橙、黑
- 点会自动保存，可以通过名称引用

## 文件说明

- `index.html` - 主程序文件
- `test-综合图形.png` - 综合功能测试截图
- `test-颜色测试.png` - 颜色功能测试截图

## 使用方法

用浏览器打开 `index.html`，在输入框中输入命令，按回车键执行。

### 命令示例

**基础图形：**
- `矩形 A(100,100) B(300,100) C(300,300) D(100,300) 蓝`
- `直线 A(100,100) B(300,300) 绿`
- `圆 O(400,300) 100 红`
- `点 N(200,200) 紫`

**相对位置：**
- `中点 A B M` - M是A和B的中点
- `分点 A B N 1/3` - N是A到B的1/3分点
- `直线 A B` - 使用已定义的点A和B画直线

**其他：**
- `清空` - 清空画布

# 需求详情：高级功能

## 修改角度
**功能描述**：指定三角形一个顶点移动，使指定角度达到目标值，保持其他边不动。

**命令格式**：
```
移动a 使角abc=90
```

**参数说明**：
- `移动a`：移动的顶点
- `角abc`：角度名称（顶点为b，由边ba和bc构成）
- `=90`：目标角度值（度）

**示例**：
```
三角形abc
移动a 使角abc=90
```
效果：移动点a，使角abc变为90度，bc边保持不动。

## 函数曲线绘制
### 一次函数
```
函数 y=ax+b
函数一次 a b
```
**示例**：
```
函数 y=2x+1
函数一次 2 1
```

### 二次函数
```
函数 y=ax²+bx+c
函数二次 a b c
```
**示例**：
```
函数 y=x²-3x+2
函数二次 1 -3 2
```

## 交点自动生成
### 交点检测触发场景
1. 画线 - 新线与现有线/圆/函数的交点
2. 画角 - 角的两条边与现有线/圆/函数的交点
3. 画圆 - 新圆与现有线/圆/函数的交点
4. 旋转/对称/平移 - 变换后图形与现有图形的交点

### 交点命名规则
- 自动递增命名：P1、P2、P3...
- 避免与手动定义的点名冲突

### 交点记录信息
- 交点坐标
- 交点名称
- 参与相交的图形（如：线AB、线CD）

### 示例
```
直线ab
直线cd
(自动生成交点P1)
```

# 需求详情：基础图形绘制

## 功能描述
帮助学生快速绘制几何图形，支持自动布局和颜色填充控制。

## 支持的图形类型
- **矩形/长方形**：`矩形abcd` 或 `矩形 A(100,100) B(300,100) C(300,300) D(100,300)`
- **直线**：`直线ab` 或 `直线 A(100,100) B(300,300)`
- **线段**：`线段ab`
- **角**：`角abc` 或 `角 A(100,100) B(200,100) C(150,200)` (A起点，B顶点，C终点)
- **圆**：`圆O 100` (指定半径) 或 `圆e ab` (使用线段长度作为半径)
- **三角形**：`三角形abc`
- **点**：`点 A(100,100)`

## 自动布局
- 不指定坐标时，图形自动居中于画布1/4区域
- 例：`矩形abcd` 自动生成居中矩形

## 颜色控制
### 全局颜色设置
- 命令：`颜色为红/蓝/绿/黄/紫/橙/黑`
- 默认颜色：黑色

### 局部颜色设置
- 命令中直接指定：`矩形abcd 红`

## 填充控制
- 默认不填充
- 关键字"填充"/"fill"启用填充：`矩形abcd 填充`

## 相对位置计算
- **中点**：`中点ab c` (c是a和b的中点)
- **分点**：`分点ab f 1/3` (f是a到b的1/3分点)
- **复用点**：已定义的点通过名称引用，不重复绘制

## 清空画布
```
清空
```
清除所有图形和函数曲线

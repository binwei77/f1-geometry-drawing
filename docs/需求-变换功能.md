# 需求详情：变换功能

## 旋转
**命令格式**：
```
旋转 三角形abc 绕o 45
旋转 三角形abc 绕o -30
```

**参数说明**：
- 正数：顺时针旋转
- 负数：逆时针旋转
- 旋转后自动命名为A'B'C'形式
- 原图形保留

**示例**：
```
三角形abc
旋转 三角形abc 绕a 90
```
效果：三角形abc绕点a顺时针旋转90度，生成三角形a'b'c'

## 对折（对称）
**命令格式**：
```
对称 三角形abc 关于直线l
对折 三角形abc 关于直线l
```

**参数说明**：
- 对称后的图形自动命名为A'B'C'形式
- 原图形保留

**示例**：
```
三角形abc
直线de
对称 三角形abc 关于直线de
```

## 平移
**命令格式**：
```
平移 三角形abc 使a到m
```

**参数说明**：
- 平移后的图形自动命名为A'B'C'形式
- 原图形保留

**示例**：
```
三角形abc
点m
平移 三角形abc 使a到m
```
效果：将三角形abc的a点移动到m点，整个三角形跟随移动

# 需求详情：删除功能

## 功能描述
提供灵活的图形删除能力，支持按类型、名称、点等多种条件删除图形，并支持撤销删除操作。

## 列出图形命令
**命令格式**：
```
列表
list
```

**输出格式**：
```
图形清单 (共5个)：
1. 矩形abcd - 点：a,b,c,d
2. 线段cd - 点：c,d
3. 中点ab-c - 点：a,b,c
```

## 按类型删除
**命令格式**：
```
删除矩形
删除线段
删除圆
删除中点
删除分点
删除旋转
删除对称
```

**示例**：
```
删除矩形  # 删除所有矩形，保留其他图形
```

## 按名称删除
**命令格式**：
```
删除矩形abcd
删除中点ab-c
```

**示例**：
```
矩形abcd; 中点ab c; 中点bc d
删除矩形abcd  # 删除矩形，保留中点c和d
```

## 按点删除
**命令格式**：
```
删除点a
```

**示例**：
```
矩形abcd; 中点ab c; 中点bc d
删除点a  # 删除矩形abcd和中点ab-c，保留中点bc-d
```

## 删除最新
**命令格式**：
```
删除最新的
```

## 撤销删除
删除操作支持撤销，恢复被删除的图形。

```
矩形abcd; 线段ab
删除线段
撤销  # 线段ab恢复
```

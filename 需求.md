# F1几何绘图项目需求

## 项目概述
帮助学生快速绘制几何图形和函数曲线，全程通过文字命令完成，无需鼠标操作。移动优先设计，支持自动布局。

## 功能模块

### 1. 基础图形绘制
- 支持矩形、直线、线段、角、圆、三角形、点
- 自动布局：不指定坐标时图形自动居中
- 颜色控制：全局和局部颜色设置，默认黑色
- 填充控制：默认不填充，"填充"/"fill"关键字启用
- 相对位置计算：中点、分点
- 清空画布

📄 详细内容：[需求-基础图形.md](docs/需求-基础图形.md)

### 2. 变换功能
- 旋转：围绕指定点旋转，正数顺时针，负数逆时针
- 对折/对称：关于直线对称
- 平移：指定点移动到目标位置
- 撤销/重做：操作历史管理

📄 详细内容：[需求-变换功能.md](docs/需求-变换功能.md)

### 3. 高级功能
- 修改角度：移动顶点使角度达到目标值
- 函数曲线绘制：一次函数、二次函数
- 交点自动生成：检测并标注线与线、线与圆、圆与圆、函数与图形的交点

📄 详细内容：[需求-高级功能.md](docs/需求-高级功能.md)

### 4. 删除功能
- 列出所有图形
- 按类型删除：删除矩形、线段、圆等
- 按名称删除：删除指定名称的图形
- 按点删除：删除包含指定点的所有图形
- 删除最新图形
- 支持撤销删除

📄 详细内容：[需求-删除功能.md](docs/需求-删除功能.md)

## 画布配置
- 网格背景：5px间距，#cccccc色，1px线宽
- 无中心轴线（不显示四象限分隔线）
- 缩放功能：支持放大/缩小按钮
- 坐标系：显示坐标网格，原点在画布中心

## 技术要求
- HTML5 Canvas + 原生JavaScript，无外部依赖
- 支持主流浏览器和移动端
- 命令执行响应时间 < 100ms
- 支持复杂图形组合（50+图形元素）
